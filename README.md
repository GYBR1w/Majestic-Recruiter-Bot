# 🏰 Majestic Recruiter Bot (Система приема заявок в RP-семью)

**Автоматизированный Discord-бот для профессионального рекрутинга и управления заявками на RP-серверах (на основе Majestic/GTA 5 RP).**

Этот бот полностью автоматизирует процесс найма, избавляя рекрутеров от рутинной работы с анкетами, тредами и уведомлениями.

---

## 🎯 Ключевые возможности

* **Модальные формы:** Сбор данных о кандидатах, включая обязательный **Статический ID**.
* **Приватные треды:** Автоматическое создание закрытого чата с кандидатом и оповещением хайрангов.
* **Интерактивный лог:** Отправка анкет в специальный канал с кнопками **"Принять"** и **"Отклонить"** для мгновенного ответа.
* **Контроль Staff:** Команда `/close_application` для моментального закрытия и архивирования активной ветки.
* **Защита от спама:** Встроенный кулдаун на подачу повторных заявок.
